# tc_script
Telechips Singapore Branch's Scripts 
1) fwdn_script.bat : FWDN(firmware downloader) - Windows OS
2) fastboot_script.bat : Bootloader / Kernel / Framework / Partition update - Windows OS
3) build_script.sh - Linux OS
